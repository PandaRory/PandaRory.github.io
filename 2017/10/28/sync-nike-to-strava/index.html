Sync Nike+ to Strava

Posted on 2017-10-28 Edited on 2024-01-31 In Note

跑跑跑~

如 500 天瘦 70KG 的 Izero 所說，有 log 才能 debug，跑步要有紀錄才有動力😁

因為公司的同事都是使用 Strava，想說就跟著用，但是又想保留之前 Nike+ Run 的記錄，所以找到 https://nike.bullrox.net，操作頁面很簡單，登入 Nike+ 和 Strava 後，就會出現按鈕可以直接同步。網路上有很多做法和應用程式可以使用，但是好像因為 API 的更新，有些工具已經不能用了，所以記錄一下目前自己使用的方法。

感覺養成習慣後，入 Garmin 的坑就不遠了…

參考資料：https://support.strava.com/hc/en-us/articles/216917847-How-to-sync-or-transfer-from-Nike-to-Strava?mobile_site=true
