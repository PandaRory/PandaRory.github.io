[laravel/framework] Factory depends on Finder

Posted on 2020-03-24 Edited on 2024-01-31 In Note

Issue #32070 Factory depends on Symfony\Component\Finder\Finder

第一次在 Open Source Project 發 issue，是 PHP 最受歡迎的框架之一：Laravel。

Laravel 的作者有將這個 Framework 的各個 components 拆出來到其他 repo，讓其他人可以獨立使用，然後我有一個 project 需要使用 lluminate/database 這個 component 來做 Unit Test 的資料遷移（製造假資料），發現其中 Factory 依賴 Symfony\Component\Finder\Finder，但是卻沒有記在 composer.json，因此發了這個 issue 告知作者應該將 Finder 列在 suggest。

P.S. lluminate/database 是不能發 issue 的，只能發在 laravel/framework

可以看到作者很快的回覆了，也迅速地送出 PR 修正，成功完成一個被接受的 issue，為 Laravel 貢獻一點點點點的心力。
